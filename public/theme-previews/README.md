# Тема-превюта за onboarding wizard-а

Скрийншоти на началните страници на 9-те демо магазина (viewport 1200×760,
JPEG, resize до 600px ширина). Ползват се от
`src/components/dashboard/website/wizard/theme-preview-card.tsx`.

**Регенериране при редизайн на тема** (нужен работещ dev сървър + seed-нати
демота): screenshot на `http://localhost:3000/s/{demo-slug}` на 1200×760 →
resize 600px / quality 72 → `{themeId}.jpg` тук. Демо slug-овете са в
`scripts/seed-demo-shops.mjs` (atelie-glina, vitrina-moda, puls-streetwear,
efir-kozmetika, oniks-luks, signal-tehnika, osnova-stroiteli,
granit-instrumenti, klasik-za-doma).
