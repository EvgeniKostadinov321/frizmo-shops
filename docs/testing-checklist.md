  проверено в базата). Пътьом: добавени пояснения **тегло↔количество** + `inputMode` на числовите
  полета (product-form / variants / ръчна поръчка) — ⚠️ **НЕ commit-нато още** (чака `pnpm check`).
- ✅ **№2 Product feed** — минат: валиден RSS/XML, 9 продукта, `g:sale_price` (4 промо) /
  `shipping_weight` / `availability` (in/out_of_stock) / `brand` / `condition=new` /
  `identifier_exists=no` — коректни; Copy бутон в `/dashboard/store`. (Opera GX отваря feed-а в
  вграден RSS четец — нормално; суров XML → Ctrl+U / Chrome.)
- ✅ **№3 Abandoned cart** — минат **ИЗЦЯЛО end-to-end**: opt-in запис (pending, snapshot 115€) →
  cron гардиран → **имейл в пощата** („Забрави ли нещо в количката?") → `sent`; **401 без
  CRON_SECRET**; втори cron → `processed:0` (без повторение). Остава само ops: **`CRON_SECRET`
  във Vercel prod + Redeploy**.
- ✅ **№4 Провери поръчка** — валиден / грешен телефон / несъществуващ номер / rate-limit ✅.
- ✅ **№5 Изтриване на акаунт** — пълно изтриване на реален втори акаунт, потвърдено в базата.
- ✅ **№6 Конверсионно трио** — търсене (+ визуални фиксове) / доставка / trust badges ✅.
- ✅ **№7 Доверие+AOV** — verified бадж (реален match) / cross-sell / „Нов" badge ✅.

**🎉 ВСИЧКИТЕ 7 МИНАХА.** Пътьом оправени визуални бъгове (двоен native search „×", мобилно
