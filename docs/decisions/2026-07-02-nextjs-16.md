# ADR: Next.js 16 вместо Next.js 15

**Дата:** 2026-07-02 · **Статус:** прието

**Контекст:** Спецификацията предвиждаше Next.js 15, но `create-next-app@latest` вече scaffold-ва Next.js 16.2 (актуалният stable). App Router моделът (Server Components, Server Actions, ISR) е същият; Turbopack е default.

**Решение:** Оставаме на Next.js 16 — нов проект се строи на текущия major; downgrade би ни лишил от support прозореца. Всички документи са обновени.

**Последици:** При въпроси за API-та проверяваме вградените docs (`node_modules/next/dist/docs/`, виж AGENTS.md) или Context7 — Next 16 има breaking changes спрямо training data.
